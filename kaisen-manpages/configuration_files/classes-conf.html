    NOM

       classes.conf - Fichier de configuration des classes de CUPS

    DESCRIPTION

       Le  fichier  classes.conf  définit les classes d'imprimantes locales disponibles. Il est habituellement situé dans le répertoire /etc/cups et est tenu à jour
   par le programme cupsd(8). Ce fichier n’est pas prévu pour être édité ou traité manuellement.

    NOTES

       Le nom, l’emplacement et le format de ce fichier sont des détails d’implémentation qui changeront dans les prochaines versions de CUPS.

    VOIR AUSSI

       cupsd(8), cupsd.conf(5), mime.convs(5), mime.types(5), printers.conf(5), subscriptions.conf(5), aide en ligne de CUPS : <URL:http://localhost:631/help>)
